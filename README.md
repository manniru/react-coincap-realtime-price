## Realtime cryptocurrency price update using coincap API.

![Realtime cryptocurrency price update screen shot](/realtime-crypto-price-coincap.png?raw=true "Realtime cryptocurrency price update screen shot")

Technology Stack:
1. Reactjs
2. Websocket
3. Coincap API
4. React Material UI

#### Setup:
```
npm install
```
#### Development:

```
npm start
```

### Production:
```
npm run build
```
